# Unity Services System

Service locator and loader with editor tools

## Installation

Add this package to your Unity project using the UPM (Unity Package Manager).

### Git URL

Latest version:

```text
https://github.com/juanpablomaggi/GameTools-UnityServicesSystem.git
```

Specific version:

```text
https://github.com/juanpablomaggi/GameTools-UnityServicesSystem.git#1.0.1
```

### Steps

1. Open your project in Unity
2. Open **Window > Package Manager**
3. Click the **+** button and select **Add package from git URL...**
4. Paste: `https://github.com/juanpablomaggi/GameTools-UnityServicesSystem.git`
5. Click **Add**
